利用可能なコマンド一覧

25コマンド
9カテゴリ
16全員使用可能
9管理者専用
共通コマンド
/help 全員
コマンド一覧とヘルプを表示します。
/ping 全員
Botの応答速度とWebSocket遅延を確認します。
/serverinfo 全員
サーバーのメンバー数・チャンネル数・ブースト情報などを表示します。
/userinfo 全員
ユーザーのロール・参加日・IDなどの情報を表示します。対象省略時は自分の情報を表示。
管理コマンド
/ban 管理者
ユーザーをBANします。メッセージ削除日数（0〜7日）の指定が可能です。
/kick 管理者
ユーザーをサーバーからキックします。
/timeout 管理者
ユーザーをタイムアウトします。60秒〜1週間の期間を選択できます。
/admin Bot管理者
Botの詳細ステータス確認・全サーバーへのブロードキャスト送信などの管理操作を行います。
経済システム
/balance 全員
残高とサーバー内順位を確認します。対象ユーザーを指定して他の人の残高も確認可能。
/daily 全員
デイリー報酬（200コイン）を受け取ります。毎日日本時間0:00にリセットされます。
/work 全員
仕事をしてコインを稼ぎます。10分のクールダウンがあり、50〜150コインをランダムで獲得。
/pay 全員
他のユーザーにコインを送金します。残高不足の場合はエラーが表示されます。
/shop 全員
listで商品一覧を表示、buyでコインを使って商品を購入できます。
ゲーム・ランキング
/slot 全員
スロットマシンで遊びます。10〜1000コインを賭けて最大50倍の払い戻しを狙えます。
/omikuji 全員
1日1回おみくじを引けます。大吉〜大凶の7段階で、結果に応じてコインボーナスがあります。日本時間0:00リセット。
/leaderboard 全員
サーバーのランキングを表示します。コイン残高・累計獲得・ログイン連続日数の3種類に対応。
ログインボーナス
/loginbonus check 全員
自分のログインボーナス状況と次のマイルストーンまでの残り日数を確認します。
/loginbonus info 全員
次回リセット日時とボーナス報酬一覧（連続日数別）を確認します。
/loginbonus ranking 全員
サーバー内のログイン連続日数ランキングを表示します。
グローバルチャット
/globalchat setup 管理者
このチャンネルをグローバルチャットに設定します。他のサーバーのメッセージが届くようになります。
/globalchat remove 管理者
このチャンネルのグローバルチャット設定を解除します。
/globalchat list 管理者
現在グローバルチャットに接続しているサーバー一覧を表示します。
ロールパネル・固定メッセージ・チケット
/rolepanel create 管理者
セレクトメニュー式のロールパネルを作成します。ロールIDをカンマ区切りで指定。
/rolepanel delete / list 管理者
パネルの削除と一覧確認ができます。
/pin set 管理者
チャンネルに固定メッセージを設定します。既存の固定メッセージは自動で削除されます。
/pin info / remove 管理者
現在の固定メッセージ情報の確認と削除ができます。
/ticket setup 管理者
チケットパネルを設置します。ボタンを押すとプライベートチャンネルが自動作成されます。
/ticket delete 管理者
チケットパネルを削除します。
VC・音声
/vchub create 管理者
VCハブを作成します。参加するだけで個人VCが自動生成され、退出すると自動削除されます。
/vchub delete / list 管理者
VCハブの削除と一覧確認ができます。
/voice join 全員
VCに参加して読み上げを開始します。チャンネルを省略すると現在参加中のVCに接続します。
/voice leave 全員
VCから退出して読み上げを停止します。
/voice setup / remove 管理者
自動参加VCの設定・解除ができます。指定したVCに誰かが参加するとBotが自動で参加します。
/vcmodpanel 管理者
VCモデレーションパネルを作成します。ミュート・デフ・キック・ロック操作がボタンで行えます。
ユーティリティ
/upreminder setup 管理者
Fortify BotのUPを検知して通知します。1時間後に再度リマインドします。メンションするロールを指定可能。
/upreminder remove / status 管理者
UPリマインダーの解除と現在の設定確認ができます。
